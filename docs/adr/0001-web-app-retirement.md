# ADR 0001: Keep the hosted app while skill parity is unproven

Status: pending paired evaluation and user workflow evidence.

## Context

The skill package now provides the planned local-first artifact contracts and five-command workflow. That does not prove that it matches or exceeds the hosted application's personalization, quality, or usability.

## Decision

Continue the dual product. Do not delete, archive, or migrate the hosted application based on implementation effort alone.

## Consequences

Run the paired evaluation and user workflow protocols before revisiting this decision. A future ADR may choose skill-first only, skill-first plus a lightweight demo site, or continued dual product, but must cite raw artifacts, scored evidence, observed user behavior, and explicit product-owner approval.
