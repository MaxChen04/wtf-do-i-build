# Skill discovery policy

Detect Vercel `find-skills` first. If it is unavailable, use `npx skills find <specific query>` when network access permits. Discovery never blocks setup, brainstorming, or local project creation.

Before recommending a candidate, inspect its `SKILL.md`, referenced files, requested permissions, source, license, maintenance signal, and fit with the approved PRD. Record the source and version or commit when available. Ask before installing any recommendation.
