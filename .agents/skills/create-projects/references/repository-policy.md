# Local repository policy

Each approved PRD creates exactly one independent local Git repository. Before mutation, show the parent directory, folder name, repository name, collision result, required files, and first commit boundary. If a folder already contains a compatible partial setup, converge on it; never overwrite unrelated work or duplicate commits.

Required contents: `PRD.md`, `README.md`, `AGENTS.md` or host equivalent, suitable `.gitignore`, selected-skill lock metadata, and project-local `build-project`. Add `CONTEXT.md` only when the project needs literal-purpose domain vocabulary. Verify links and the PRD before committing.
