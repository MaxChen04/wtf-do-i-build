# Behavior scoring

Evaluate the process, not exact prose. A passing host run demonstrates the named behavior with an inspectable artifact and does not violate the listed guardrails. Record the host, model tier, source availability, questions asked, selected direction, PRD result, validation output, and resume result.

For parity with the hosted application, score personalization, specificity, candidate diversity, unsupported inference, demand evidence, PRD executability, grounding, and clean-session resumability side by side. Model prose may differ; gates and artifact structure may not.
