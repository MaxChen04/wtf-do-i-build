# Signal rules

Use stated answers and recent free text as intent. Use résumé, GitHub, and historical work as capability, access, and observed behavior. The precedence order is: stated current intent, current free text, recent résumé, then old or generic context. Split a dual-purpose signal instead of averaging it.

An inference must be labelled, bounded, and confidence-limited. It cannot become an intent or passion claim. Down-weight weak evidence; do not erase it. Reject a candidate that would still fit a generic person with the same job title.

Memory can establish a remembered preference, repeated question, prior decision, or observed friction. Memory cannot originate intent. When it conflicts with what the user says now, preserve the conflict and follow the current statement.
