# Resumability

Freeze IDs once emitted. Add inserted work through suffixes rather than renumbering existing steps. Use only `[ ]`, `[~]`, `[x]`, and `[!]`. A fresh agent reads the whole document, trusts checkboxes over prose, and resumes the first non-`[x]` step.

Keep the `## ▸ Resume here` header current: status, first unfinished step, immediate verification target, and links to any nested specifications. Long detail belongs in a linked nested document rather than a bloated parent PRD.
